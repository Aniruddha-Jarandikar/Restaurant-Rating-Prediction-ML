# Restaurant Rating Prediction

## Project Overview
This project predicts restaurant ratings using Machine Learning regression models.

## Dataset
Restaurant dataset containing location, cuisine, votes, pricing, and delivery-related information.

## Models Used
- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor
- Gradient Boosting Regressor

## Evaluation Metrics
- R² Score
- RMSE
- MAE

## Results

| Model | R² Score |
|---------|---------|
| Linear Regression | 0.4095 |
| Decision Tree | 0.3025 |
| Random Forest | 0.6333 |
| Gradient Boosting | 0.6347 |

## Best Model
Gradient Boosting Regressor achieved the highest performance with an R² score of 0.6347.

## Feature Importance
Votes was identified as the most influential feature affecting restaurant ratings.
